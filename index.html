Flow as below, Tool Used: AWS-EC2-Ubuntu ,Git,GitHUB, Jenkins, Docker, Ngnix

Installed & configured Git on AWS-Ubuntu -EC2 instance

Created index.html

I Pushed Index.html on GitHUB via Git (Local Repo)

Configured master repo on GitHub & configured webhook for Jenkins on GitHUB

Installed & configured Jenkins on AWS. Also, Jenkins integrated with GitHUB

Created Project-(JOB) on Jenkins with Shell-Script

As per script & job configuration setting, when file/code push into GitHub.then, automatically Jenkins JOB will Trigger

In that JOB, some tasks are assigned : Install Docker (Publish on Internet) & install ngnix on AWS-Slave server . Also Index.html file will move automatically ngnix particular path

For installation Docker ,Docker file (script) created.
